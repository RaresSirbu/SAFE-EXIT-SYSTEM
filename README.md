Acest proiect prezintă dezvoltarea unui asistent de siguranță pentru deschiderea portierelor, realizat pe baza capabilităților platformei Arduino. Sistemul monitorizează continuu proximitatea pietonilor, bicicliștilor sau a altor autovehicule și gestionează blocarea automată a ușilor pentru a preveni accidentele.

Logica de control, implementată în C++, permite evaluarea în timp real a riscului și menținerea ușilor blocate până când zona devine sigură, oferind o protecție eficientă și fiabilă pentru utilizatori.

Proiectul integrează senzori ultrasonici, servo-motoare și elemente de control logic, asigurând o interacțiune sigură și coerentă între utilizator și mediu. Scopul principal a fost crearea unei soluții care să reducă riscul de accidente la deschiderea portierelor.
